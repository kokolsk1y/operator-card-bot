# Бот для операторов — генератор карточек

Telegram-бот **@stvinfografikabot**. Собирает карточку товара: фон → заголовок →
фото товара (авто-вырезка фона + свечение) → до 4 характеристик → готовый PNG.

Полное описание задачи, измерения макета и алгоритм — в [ТЗ.md](ТЗ.md).

## Как запустить

Из этой папки:

```bash
node index.mjs
```

Бот работает в режиме polling, пока запущен этот процесс (нужен включённый ПК
и интернет). Остановить — `Ctrl+C`.

## Как пользуется оператор

1. Открывает в Telegram **@stvinfografikabot**, жмёт `/new` (или «Старт»).
2. Шаг 1 — присылает **название товара** (заголовок).
3. Шаг 2 — присылает **фото товара**. Бот сам убирает фон (~15 сек), показывает
   превью с кнопками **✅ Дальше / 🔁 Переснять**.
4. Шаги 3–6 — характеристики (до 4, не больше 20 символов каждая):
   1-я → оранжевый квадрат, остальные → полоски. Любую можно **⏭ Пропустить**
   или сразу **✅ Собрать карточку**.
5. Получает готовый PNG. Кнопками можно поправить **заголовок / фото /
   характеристики**, **📐 подвинуть элементы** или начать **новую карточку** —
   без рестарта с нуля.

### 📐 Режим «Подвинуть»

Кнопка **📐 Подвинуть** под готовой карточкой открывает панель сдвига:
- **◀ ▶** — выбрать элемент: заголовок, бейдж, оранжевый блок, **вся группа**
  (оранж + характеристики), фото товара, **каждая строка-характеристика отдельно**;
- стрелки **⬆️⬇️⬅️➡️** двигают выбранный элемент;
- **Шаг** — 5 / 10 / 20 px;
- **✅ Готово** — сохранить и получить финальный PNG.

Превью обновляется на каждый сдвиг. Смещения хранятся в сессии (`offsets`).

## Файлы

| Файл | Назначение |
|---|---|
| `index.mjs` | Бот + диалог (grammY) |
| `render.mjs` | Сборка карточки (HTML + Puppeteer → PNG). **Все зоны макета — в `LAYOUT` вверху файла.** |
| `cutout.py` | Удаление фона (rembg/birefnet) + свечение края + проверка качества |
| `config.json` | Токен бота (СЕКРЕТ, в `.gitignore`), модель, размеры |
| `фон.png` | Фиксированный фон 900×1200 |
| `пример карточки.png` | Эталон макета |
| `test-render.mjs` | Локальный тест рендера без Telegram: `node test-render.mjs путь_к_вырезанному.png` |

## Зависимости

- Node.js + `grammy`, `puppeteer`, `sharp` (уже в `node_modules` проекта)
- Python + `rembg`, `numpy`, `Pillow`, `scipy` (уже установлены)
- Модель `birefnet-general.onnx` (~1 ГБ) — уже в `~/.u2net/`

## Настройка макета

Если нужно подвинуть зоны (отступы, размеры) — правится **только** объект
`LAYOUT` в начале `render.mjs` (координаты в px при ширине 900). После правки:
`node test-render.mjs .tmp/cut_test.png` и сравнить с `пример карточки.png`.
